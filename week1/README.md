## Day 1 — First real Gemini API call

- File: `hello_gemini.py`
- Asked Gemini to explain what an AI PM does differently from a regular PM.
- Verified end-to-end: Python venv + .env loading + google-genai SDK + Gemini 2.5 Flash.
- Output: see `day1_screenshot.png`

### Key takeaway from Gemini's response
The response highlighted that AI PMs work with **probabilistic outputs**, **model drift**, **bias**, and **evals frameworks** — none of which a generalist PM typically owns. This becomes the through-line for the next 10 weeks.
